# Danas.rs News Portal - Frontend Recreation 📰

A high-fidelity, pixel-perfect recreation of the **Danas.rs** news portal. This project was developed as part of the **"Web Design and Development"** course at **RAF (Računarski Fakultet)** in Belgrade.All rights to the original design and content belong to Danas.rs.

## 🚀 Project Overview
The goal of this project was to master complex UI layouts and responsive web design by recreating a real-world, content-heavy news platform.

### Key Features:
* **Massive Custom CSS:** Over **4,000 lines of custom CSS** written from scratch to ensure every component matches the original design.
* **Advanced Styling:** Includes complex hover effects, transitions, and deeply nested selectors for high-fidelity UI recreation.
* **Comprehensive Page Coverage:** Recreated not just the home page, but also Politics, Economy, Climate Change, Rule of Law, and more, ensuring a consistent user experience across the entire site.
* **Full Responsiveness:** Optimized for all screen sizes using CSS Media Queries.
* **Multi-page Architecture:** Deep linking between sections like Belgrade, Novi Sad, and Kragujevac.

## 🛠️ Technologies Used
* **HTML5:** Semantic structure for news articles and navigation.
* **CSS3:** Advanced layouts using **Flexbox** and **CSS Grid**, custom fonts, and FontAwesome icons.
* **Git:** Version control and deployment via GitHub.
